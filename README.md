# TradeWise

TradeWise is a stock trading simulation platform built as a full-stack academic project. It provides a realistic trading workflow with portfolio tracking, watchlist management, transaction history, and a DevOps-enabled deployment and monitoring setup.

## Features

- User registration and login
- Dashboard for user overview
- Market page for stock browsing
- Buy stock simulation
- Portfolio tracking
- Transaction history
- Watchlist management
- MySQL database integration
- Spring Boot backend APIs
- Frontend using HTML, CSS, and JavaScript
- Dockerized application setup
- Monitoring with Spring Boot Actuator, Prometheus, and Grafana
- CI pipeline using Jenkins
- Kubernetes deployment and service manifests
